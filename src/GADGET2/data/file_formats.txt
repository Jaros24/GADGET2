Each line in the flatlookup table files are formatted as follows: Cobo, ASAD, AGET, CHANNEL, PAD

==h5 files generated by merging software==
[meta][meta] has first and last event number and timestamps
['get'][evt#_data] contains data from individual events, formatted as an N by 517 array, where N is the number of pads that fired. The first 5 entries in each of the N arrays are: Cobo, ASAD, AGET, channel, pad. The other 512 are the time sequence of ADC counts.

==padxy.txt==
Each line number corresponds to a pad, and the contents of the line are x,y position in mm
